# 巴蜀文化双语主持教学智能体 - 微信小程序版

这是现有网页项目的微信小程序版本，当前已实现：

- 首页导航：文字测评、录音专项测评、仪表盘
- 录音专项测评选题页：展示全部巴蜀文化双语文稿选题
- 录制页面：中文录音、English Recording 分开录制、播放、上传
- 结果页面：展示总分、准确度、流畅度、完整度和训练建议

## 如何在微信开发者工具打开

1. 打开“微信开发者工具”。
2. 点击“导入项目”。
3. 项目目录选择：

```text
D:\2026年7月\chapter 1-5\bilingual-host-ai\wechat-miniprogram
```

4. AppID 可以先选择“测试号”或使用自己的小程序 AppID。
5. 导入后，打开右上角“详情” -> “本地设置”。
6. 勾选“不校验合法域名、web-view、TLS 版本以及 HTTPS 证书”。
7. 点击“编译”，进入首页后选择“录音专项测评”。

## 后端接口地址

测试接口地址在 `app.js`：

```js
apiBaseUrl: 'http://8.137.198.15/api'
```

如果你以后有自己的 HTTPS 域名，例如：

```text
https://api.your-domain.com/api
```

把 `app.js` 里的地址替换即可。

## 正式发布必须做的事

微信小程序正式发布时，不能直接使用 `http://8.137.198.15` 这种 HTTP IP 地址。

必须满足：

1. 后端绑定域名。
2. 域名备案。
3. 配置 HTTPS 证书。
4. 在微信公众平台 -> 开发管理 -> 开发设置 -> 服务器域名里，把后端域名加入 `request 合法域名`。

完成后，用户就可以在微信里长期使用录音测评。
